# tool-code-ptit

## Run Locally

Clone the project

```bash
  git clone https://github.com/Jaxx1911/tool-code-ptit.git
```

Go to the project directory

```bash
  cd tool-code-ptit
```

Go to account file

```bash
  cd ./Data/account.txt
```

Import your account into account.txt

```bash
  "username": "your-username"
  "password": "your-password"
```

Auto submit

```bash
  go run
```
